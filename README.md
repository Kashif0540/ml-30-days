# 30-Day Machine Learning Challenge 🤖

A self-paced 30-day journey to learn Machine Learning from scratch — covering Python foundations, core ML algorithms, deep learning, and real-world deployment.

---

## 📅 Roadmap

### Week 1 — Python & Math Foundations (Days 1–7)
| Day | Topic | Concepts |
|-----|-------|----------|
| 1 | Python refresher | NumPy arrays, list comprehensions |
| 2 | NumPy deep dive | Vectorized ops, broadcasting |
| 3 | Pandas basics | DataFrames, groupby, merge |
| 4 | Data visualization | Matplotlib, Seaborn |
| 5 | Linear algebra basics | Matrices, dot product, eigenvalues |
| 6 | Probability & stats | Mean, variance, distributions |
| 7 | Mini project | EDA on a real dataset (Kaggle) |

### Week 2 — Core ML Algorithms (Days 8–14)
| Day | Topic | Concepts |
|-----|-------|----------|
| 8 | Linear regression from scratch | Gradient descent, cost function |
| 9 | Logistic regression | Sigmoid, binary classification |
| 10 | Decision trees | Entropy, Gini index |
| 11 | Random forests & bagging | Ensemble methods |
| 12 | K-means clustering | Unsupervised, centroids |
| 13 | SVM basics | Hyperplane, kernel trick |
| 14 | Mini project | Titanic survival classifier |

### Week 3 — Deep Learning Intro (Days 15–21)
| Day | Topic | Concepts |
|-----|-------|----------|
| 15 | Neural network basics | Perceptron, activation functions |
| 16 | Backpropagation | Chain rule, weight updates |
| 17 | PyTorch intro | Tensors, autograd |
| 18 | Build a basic NN | MNIST digit classifier |
| 19 | CNNs | Convolutions, pooling, filters |
| 20 | Overfitting & regularization | Dropout, L2, early stopping |
| 21 | Mini project | Image classifier (CIFAR-10) |

### Week 4 — Real Projects & Deployment (Days 22–30)
| Day | Topic | Concepts |
|-----|-------|----------|
| 22 | NLP basics | Tokenization, TF-IDF, word2vec |
| 23 | Sentiment analysis | Sklearn pipeline, text features |
| 24 | Intro to transformers | BERT, Hugging Face basics |
| 25 | Model evaluation | Precision, recall, ROC curve |
| 26 | Feature engineering | Encoding, scaling, selection |
| 27 | Save & load models | Pickle, joblib, ONNX |
| 28 | Build a Flask ML API | REST endpoint for predictions |
| 29 | Write clean ML notebooks | Structure, markdown, GitHub |
| 30 | Final project | End-to-end ML app |
